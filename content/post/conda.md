## Tuna Archive下载conda

* https://mirrors.tuna.tsinghua.edu.cn/anaconda/archive/

* 下载后用静默方式安装:

  ```bash
  bash ./<anaconda安装cli>.sh -b -p ~/anaconda3
  ```

* 激活conda:

  ```bash
  ~/anaconda3/bin/conda init
  source ~/.bashrc
  ```
